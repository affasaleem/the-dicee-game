# 🎲 Roll Royale

A modern and responsive Dice Game built using **HTML, CSS, and JavaScript**.

Players roll a virtual dice, and the game automatically determines the winner based on the highest number rolled. This project was created to practice JavaScript fundamentals, DOM manipulation, and responsive web design.

---

## 🚀 Features

- 🎲 Random dice generation (1-6)
- 🏆 Automatic winner detection
- 🔄 Dynamic dice image updates
- ✨ Modern glassmorphism UI
- 📱 Fully responsive design
- 🎨 Animated gradients and hover effects
- ⚡ Beginner-friendly JavaScript logic

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

1. Open the project in a web browser.
2. The dice are rolled automatically when the page loads.
3. The player with the higher dice value wins.
4. Click the **Roll Again** button to generate a new result.
